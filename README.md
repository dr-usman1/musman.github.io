# musman.github.io

Personal website for Dr. Muhammad Usman

## Live Website

Visit the website at: https://dr-usman1.github.io/musman.github.io/

## Features

- Professional home page with introduction
- Responsive design (works on desktop, tablet, and mobile)
- Profile picture section (placeholder with initials, customizable)
- Social media links
- Modern gradient design

## Customization

### Adding Your Profile Picture

1. Add your profile picture to `assets/images/` directory as `profile.jpg` or `profile.png`
2. Update `index.html`:
   - Find the line with `<div class="profile-picture">MU</div>`
   - Replace with: `<img src="assets/images/profile.jpg" alt="Profile Picture" class="profile-picture">`

See `assets/images/README.md` for more details.

### Updating Your Information

Edit `index.html` to customize:
- Your name and title
- Introduction text
- Social media links (GitHub, Email, LinkedIn)
- Contact information
